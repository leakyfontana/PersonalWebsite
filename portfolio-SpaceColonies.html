Data Structures: Space Colonies

Java, JavaFX and Eclipse

Space Colonies was another assigned school project. The goal was to create a program with a corresponding User Interface that simulated a pretend scenario where the Government is running a program to select candidates for a space program. The applicants were provided in a CSV input file with information regarding their name, skills scores and desired planet. If their skills met their desired planets minimum skill scores they would be accepted, otherwise they would be rejected.

To implement the solution, I utilized the Java programming language, Eclipse IDE, various data structures and JavaFX to develop the User Interface. This project helped me further understand the tasks and difficulties of all levels of the software development cycle.
